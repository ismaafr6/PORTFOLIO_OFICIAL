# IFR | Professional Portfolio Elite

Professional portfolio built with a focus on luxury UI/UX, smooth animations, and interactive elements.

## 🚀 Live Demo
[Visualizar Portfolio](https://portfolio-oficial-beta.vercel.app)

## ✨ Features

- **Luxury Motion**: 3D Tilt effects on project cards and magnetic interaction on buttons.
- **Brand Identity**: Custom branding with 'IFR' monogram and elite preloader.
- **High-End UI**: Pure Dark Mode monochromatic aesthetic with glassmorphism effects.
- **Interactive Skills**: Animated skill bars and featured section for AI & Innovation.
- **Success States**: Integrated Canvas Confetti on successful form submissions.
- **Custom Mapping**: Minimalist Toledo map with custom waypoint-based path (Cuerva -> Toledo).

## 🛠️ Built With

- **HTML5 / CSS3**: Custom CSS variables and modern grid/flexbox layouts.
- **JavaScript (ES6+)**: Custom interaction engine (Tilt, Magnetic, Scroll Reveal).
- **Leaflet.js**: Custom map implementation.
- **Canvas Confetti**: Visual feedback for user interaction.
- **Vercel**: Automatic CI/CD deployment.
